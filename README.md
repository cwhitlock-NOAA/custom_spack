Sample configuration repo for spack 1.1

Contains simplified environments and template files to test how things build

6 directories: 
  config - various config.yml files
  env - the environment files that we're using; referred to in config/config.yml
  licenses - licenses
  modulefile_templates - modulefile tempaltes; also referred to in config/config.yml
  packages - paths to custom package.py files for fre_bronx et. al.
  scripts - custom scripts for doing utility stuff in the spack repo
